# GlobalGameJam2023

A Unity 3D game created for the Global Game Jam 2023. Experience our unique gameplay directly in your browser !!!

## Controls
- `WASD` — move
- `Right Click` — look around
- `Left Click` — fire
- `Q` — switch weapons
- `F5` — restart the game

**Objective:** Destroy the two enemy bases on the map.
